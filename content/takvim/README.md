# content/takvim/ — Çok kanallı içerik takvimi

`kads calendar` ile üretilir. Kanallar: Instagram, Facebook, TikTok, LinkedIn, X, Google İşletme.
5 konsept (creatives/) + pik saatler döndürülür. ~69 post / 30 gün.

- `icerik-takvimi.csv` — tarih/gün/saat/kanal/konsept/format/metin (insan planı).
- `postiz-takvim.csv` — Postiz/n8n-hazır (date/time/channel/content/media/concept). `kads publish`.

Yeniden üret: `kads calendar --out content/takvim` · `kads publish --out content/takvim`.
Cadence (öneri): IG 5-6/hafta, TikTok 3, FB 3, X 3, LinkedIn 1-2, Google İşletme 1-2.
Sezon: Haz-Eyl manzara/kahvaltı/evcil öne; Kas-Mar hafta içi kaçış/WhatsApp öne.
