# content/ — SEO içerik brief'leri (markalı + yerel sıralama)

"kozbeyli" ve çevresi sorgularında marka otoritesi + uzun kuyruk için içerik brief'leri.
Her brief: hedef kelime, niyet, başlık/meta, H2 planı, iç linkler, kelime sayısı, CTA.
`.com` sitesine `/deneyimler` veya `/blog` altında yayınla (docs/08). Görseller özgün olmalı.
