GTM Kurulumu — Kozbeyli Konağı

Bu sayfayı sitenizin (www.kozbeylikonagi.com) geliştiricisine verin. ~30 dk işdir. Bu adım bitmeden reklam yayına alınmamalı: dönüşüm (rezervasyon) ölçülemezse bütçe körlemesine harcanır.

Mevcut durum: Sitede Google Tag Manager kurulu değil. Site Next.js (istemci-render) — etiketler JavaScript ile eklenmeli. Tek container kullanılacak: GTM-KCG6B4MJ. İkinci aday GTM-MSL2FLF5 duplikat → GTM hesabında arşivle/sil (çift sayımı önler).

Yol A — Next.js component (önerilen, repoda hazır)

Depoda tracking/ConsentAndGtm.tsx + tracking/implementation/01-consent-mode-v2.html hazır. Component'i app/layout.tsx (veya _app.tsx) içine ekleyin, GTM ID = GTM-KCG6B4MJ. Consent Mode v2 (KVKK) dahil gelir.

Yol B — Ham snippet (component kullanılmıyorsa)

1. Her sayfanın <head> en üstüne (Next.js'te next/script strategy="afterInteractive"):

<!-- Google Tag Manager -->
<script>(function(w,d,s,l,i){w[l]=w[l]||[];w[l].push({'gtm.start':
new Date().getTime(),event:'gtm.js'});var f=d.getElementsByTagName(s)[0],
j=d.createElement(s),dl=l!='dataLayer'?'&l='+l:'';j.async=true;
j.src='https://www.googletagmanager.com/gtm.js?id='+i+dl;
f.parentNode.insertBefore(j,f);})(window,document,'script','dataLayer','GTM-KCG6B4MJ');</script>
<!-- End Google Tag Manager -->

2. <body> açılışına (noscript fallback):

<!-- Google Tag Manager (noscript) -->
<noscript><iframe src="https://www.googletagmanager.com/ns.html?id=GTM-KCG6B4MJ"
height="0" width="0" style="display:none;visibility:hidden"></iframe></noscript>
<!-- End Google Tag Manager (noscript) -->
ÇİFT SAYIM UYARISI: Sitede halihazırda doğrudan gömülü gtag.js (GA4/Ads) veya fbq (Meta Pixel) varsa, bunları kaldırın. Hepsi artık yalnızca GTM içinden yönetilecek. İki yerden yüklenirse veriler iki katına çıkar ve bozulur.

GTM panelinde (tagmanager.google.com, GTM-KCG6B4MJ)

  1. GA4 Configuration tag → Measurement ID G-V3R66C3MEF, trigger: All Pages.
  2. Conversion Linker tag → trigger: All Pages (Ads için şart).
  3. Google Ads Conversion → ID AW-800024713, "rezervasyon tamamlandı" tetikleyici (HMS onay sayfası / purchase event).
  4. Meta Pixel base (Custom HTML) → 1781546559309505, All Pages.
  5. Cross-domain: GA4 Config → Configure tag settings → "domains to link": kozbeylikonagi.com + hmshotel.net. GA4 admin'de hmshotel.net'i referral exclusion'a ekleyin (rezervasyon ayrı domende).
  6. begin_checkout / purchase dataLayer event'leri: tracking/implementation/04-datalayer-events.md'deki şemaya göre. HMS onay snippet: tracking/hms-confirmation-snippet.html.
  7. Submit/Publish (GTM'de yayınlamadan canlıya geçmez).

Doğrulama (yayın öncesi şart)

  1. Tag Assistant (tagassistant.google.com) → siteyi bağla → GTM-KCG6B4MJ ateşliyor mu? Tek container mı?
  2. GTM Preview ile siteyi gez → GA4 Config + Conversion Linker All Pages'te ateşliyor mu?
  3. Test rezervasyon yap → GA4 DebugView'da begin_checkout ve purchase görünüyor mu? Ads'te dönüşüm "kaydediliyor" mu?
  4. Hepsi yeşilse: config/ads-assets.yaml → gtm_status: resolved yapın; reklamlar enable edilebilir.
Bitince: ölçüm kapalı→açık olur. Ancak o zaman Google/Meta akıllı teklif öğrenebilir ve kısıtlı bütçe doğru yere gider. Sıra: (1) bu GTM kurulumu → (2) test rezervasyonla doğrulama → (3) yeni kampanyaları enable → (4) eski 7 kampanyayı durdur.
