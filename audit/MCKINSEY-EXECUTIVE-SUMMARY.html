2) Hızlı Kazanımlar (1 hafta) — sıra önemli

  1. ÖLÇÜMÜ AÇ (her şeyin önünde): GTM GTM-KCG6B4MJ içine GA4 Config G-V3R66C3MEF + Google Ads Conversion Linker/Conversion AW-800024713 → Publish → test rezervasyonuyla doğrula. Bu yeşillenene kadar kör Search/Foça Otel (₺300/gün) kampanyasını duraklat veya kıs. Rehber: golive/GTM-KURULUM.html.
  2. Meta Pixel tamamla (bedava): Events Manager → Advanced Matching AÇ (1 tık) + hmshotel.net onay sayfasına Purchase + oda sayfalarına ViewContent.
  3. Arama görünürlüğü: Search Console + Bing Webmaster doğrula → sitemap (57 URL) → IndexNow AÇ (ChatGPT/AI-alıntı ön koşulu, rakiplerin atladığı ucuz avantaj). Rehber: docs/25.
  4. Yorum hızı (en yüksek getirili ÜCRETSİZ kaldıraç): Google puanı ~4.2 vs rakipler 4.7–4.8. Son-yorum kampanyası + tüm yorumlara yanıt → 4.5+ algısı.
  5. Temiz kampanyaları yükle: yeniden üretilmiş Google Editor paketi (marka-pinli, grup-farklılaştırılmış, "13 km"siz uyumlu metin) — Paused import et, etiketler doğrulanınca enable.

3) Stratejik Girişimler (1 ay)

  1. Gerçek attribution: ölçüm 2–4 hafta veri toplayınca MMM gamma'sını gerçek harcama-tepki eğrisinden kalibre et (şu an scale-invariant sezgisel, gerçek MMM değil) + offline/telefon dönüşüm import'unu bağla.
  2. decision_memory gerçek reflection: şu an "actual_outcome" sabit-simüle (kodda açıkça işaretli). FactCampaignPerformance'tan gerçek CPA/dönüşüm oku → heuristik promote'u gerçek veriye bağla.
  3. Remarketing + Demand Gen canlıya: begin_checkout terk havuzu en sıcak retargeting — ölçüm sağlam aktıktan sonra aç.
  4. B2B Aliağa motoru: sell-sheet + outreach sekansını devreye al (kurumsal etkinlik/konaklama talebi).
  5. Executor'ı gerçeğe bağla: şu an simülasyon (lokal state). Guardrail + circuit breaker + approval kuyruğu hazır; writes-default-off ile gerçek Google Ads API yazımına kademeli geç.

4) Gelir Etkisi Tahmini MODEL — canlı ölçüme bağlı

En büyük etki yeni kampanya değil; kör harcamayı durdurmak + ölçmek. Mevcut durum: ~₺9.000/ay canlı Search harcaması, 0 ölçülen dönüşüm.

  • Verimlilik (hemen): ölçüm + marka-pin + NonBrand farklılaştırma + negatifler → mevcut ~₺9k/ay Google harcamasında tahmini %20–35 daha düşük CPA = ~₺1.800–3.150/ay tasarruf/yeniden dağıtım.
  • Gelir (atfedilebilir): Purchase takibi canlı olunca, ~1.450 tık/ay üzerinde muhafazakâr %1,5–2,5 rezervasyon dönüşümü × ~₺4.000 ort. rezervasyon → kabaca ₺85k–145k/ay görünür komisyonsuz doğrudan rezervasyon (şu an ölçülemediği için optimize edilemiyor).
  • Marka savunması: marka-terimi poz-1 pin, OTA komisyonunu (~%15–20) marka aramalarında geri kazanır.

Dürüstlük kaydı: bu sayılar MODEL tahminidir, garanti değil — canlı etiket + gerçek veriye bağlıdır. Sistem artık bunları iddia etmek yerine dürüstçe ÖLÇECEK şekilde donatıldı (sahte örnek-veri sıfırlandı, ölçüm-doğruluk kaynağı tek).
