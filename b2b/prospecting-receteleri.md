# B2B Prospecting Reçeteleri (kişi/firma bul)

## Apify (web/LinkedIn)
- LinkedIn Company/People scraper: "Aliağa" + sektör (petrokimya/demir-çelik/liman) → İK/Admin/Satınalma/Etkinlik rolleri.
- Aliağa OSB üye firma listesi (web) + Aliağa Ticaret Odası → firma + iletişim.
- MCP akışı: search-actors "LinkedIn" → fetch-actor-details → call-actor (maxItems küçük, free-tier) → get-dataset-items.

## Cowork prospecting eklentileri (varsa)
- **Apollo / ZoomInfo / Common Room / Nimble / vpai** — firma + karar verici + e-posta/telefon zenginleştirme.
  ICP: "Aliağa 50+ km, sanayi/rafineri/çelik/liman, İK veya Etkinlik veya Satınalma rolü."

## Manuel
- Aliağa Ticaret Odası üye listesi, OSB rehberi, firma "kariyer/İK" sayfaları.
- LinkedIn arama: firma + "İnsan Kaynakları" / "İdari İşler" / "Kurumsal İletişim".

## Akış
1. Firma + rol listesi → `hesap-listesi.csv`.
2. Outreach (outreach-sablonlari.md): LinkedIn bağlantı → e-posta → takip.
3. Görüşme/keşif ziyareti → çerçeve anlaşma. CRM/CSV'de durumu izle.
4. Kazanılan hesabı tekrar/upsell (etkinlik + konaklama + catering) ile büyüt.
