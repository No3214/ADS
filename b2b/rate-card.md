# Kurumsal Rate Card (yapı — fiyatları sen belirle)

> Fiyatlar örnektir; gerçek maliyet/sezon/anlaşmaya göre belirle. Kurumsal = hacim + tekrar
> karşılığı özel koşul. Bireysel BAR'dan farklı, sözleşmeli.

## Konaklama
| Ürün | Yapı |
|---|---|
| Kurumsal gece tarifesi (BB) | BAR − %X kurumsal indirim; kurumsal fatura |
| Çerçeve anlaşma (yıllık) | Hacim taahhüdüne göre kademeli indirim + öncelikli müsaitlik |
| Uzun konaklama (7+ / 30+ gece) | Yüklenici/duruş; oda blokajı + ek indirim + esnek iptal |
| Tam kiralama (buyout) | Tüm konak günlük; etkinlik/gizlilik |

## Toplantı / etkinlik
| Ürün | Yapı |
|---|---|
| Toplantı (DDR — day delegate) | Kişi başı: salon + 2 kahve arası + öğle + ekipman |
| 24 saat paketi (konaklamalı) | DDR + 1 gece + akşam yemeği |
| Gala / kurumsal yemek | Kişi başı menü (min kişi); tarihi konak + 200 kişi |
| Catering (dış mekân) | Kurumun tesisinde; kişi başı + lojistik |

## Koşullar
Kurumsal fatura, sözleşme, depozito/iptal politikası, ödeme vadesi (kurumsal), tek muhatap.
Çerçeve anlaşma şablonu hukuk onayından geçmeli.
