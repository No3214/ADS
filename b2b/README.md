# b2b/ — Kurumsal (Aliağa sanayi) gelir motoru

Kozbeyli Konağı, Türkiye'nin en yoğun sanayi havzalarından **Aliağa**'ya ~25–30 km (30–40 dk).
Aliağa = İzmir'in en büyük şirketleri (Petkim, STAR Rafineri/SOCAR, Tüpraş, demir-çelik dev
fabrikaları, gemi söküm yatları, Aliağa OSB 72 firma, Nemrut limanları). Bu kümeden **düzenli,
yüksek değerli kurumsal gelir** mümkün: yönetici konaklaması, tedarikçi/yüklenici barınması
(bakım-duruş dönemleri), toplantı/eğitim, kurumsal yemek/gala, müşteri ağırlama, catering.

Bireysel turizmden farkı: **sezon-bağımsız, tekrar eden, çerçeve anlaşmalı** (B2B düzenli gelir).
Avantajımız: tesise yakın ama sahadan uzak **sessiz, özel, premium** bir konak + restoran + 200
kişilik etkinlik alanı. Tam-kiralama (buyout) + gizlilik mümkün.

Paket: hedef firmalar (`hedef-firmalar.md`), değer önerisi, rate card, MICE paketleri, outreach
şablonları, hesap CSV, prospecting reçeteleri. Komut: `kads b2b`. Outreach: LinkedIn (Postiz) +
e-posta + prospecting (Apollo/ZoomInfo/Common Room/Apify).
