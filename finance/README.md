# finance/ — Bütçe & Öngörü Modeli

`kozbeyli-butce-ongoru.xlsx` — 5 sayfalık, formüllü (sıfır hata doğrulandı) Excel modeli:
- **Bütçe:** 30.000 TL kanal dağılımı (Google 15k + Meta 15k), toplam formülü.
- **ROAS Senaryo:** hedef ROAS (3x/4x) → gereken gelir + rezervasyon (7.500/10.000 ort.) + başabaş.
- **Aylık İzleme:** 12 ay tracker — harcama/gelir/rezervasyon gir, ROAS/CPA/toplam otomatik.
- **Sezon:** sezon bütçe ağırlığı (doluluk-duyarlı).
- **Funnel KPI:** 5 aşama çerçevesi.

Renk kodu: **mavi = girdi (değiştir)**, siyah = formül. Sayılar planlama tahminidir, garanti değildir.
