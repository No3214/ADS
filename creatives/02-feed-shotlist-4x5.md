# Feed Çekim Listesi (4:5, 1080×1350)

Tek güçlü kare veya 3-5 karelik carousel. Her konsept için:

- **K1 Taş Konak:** konak cephesi tek kare, sıcak ton; carousel: cephe → avlu → oda → teras.
- **K2 Manzara:** çatı terası + Ege manzarası 4:5; carousel: oda → teras → gün batımı.
- **K3 Kahvaltı:** sofra üstten kare; carousel: genel sofra → pişi → reçel → dibek kahvesi.
- **K4 Evcil:** avluda evcil dost karesi; carousel: dost → oda → avlu.
- **K5 Müsaitlik:** sakin oda + kısa metin "hafta içi huzur"; tek kare.

Kurallar: metin görselin %20'sini geçmesin; marka logosu köşede; net, yüksek çözünürlük;
özgün fotoğraf (stok değil). Başlık alanı için üst/alt %15 nefes payı bırak.
